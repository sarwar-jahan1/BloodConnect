# BloodConnect  
### Save Life, Be a Hero

## Project Overview

BloodConnect is a frontend-based blood donation finder website designed to help people quickly find blood donors during emergency situations. The website provides a simple, user-friendly, and responsive platform where users can search donors by blood group, view donor information, and submit emergency blood requests.

This project is developed using HTML, CSS, and JavaScript with a modern responsive user interface.

---

# Objectives

- To develop a responsive blood donation website
- To practice frontend web development skills
- To implement JavaScript DOM manipulation and filtering
- To create a user-friendly donor searching system

# Features

## Responsive Navigation Bar
- Clean navigation menu
- Mobile responsive layout
- Emergency request button

## Hero Section
- Motivational heading
- Call-to-action buttons
- Blood donation illustration

## Blood Group Filtering System
Users can filter donors according to blood groups:
- A+
- A-
- B+
- B-
- O+
- O-
- AB+
- AB-

## Donor List Section
Each donor card contains:
- Donor name
- Blood group
- Location
- Phone number
- Contact button

## Emergency Blood Request Form
Users can submit emergency blood requests by providing:
- Name
- Blood group
- Hospital/location
- Phone number
- Additional message

## Contact & Footer Section
- Contact information
- Social media links
- Quick navigation links
